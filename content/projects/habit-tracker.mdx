## Overview

Built a full-stack habit tracking platform with a responsive **React (Vite) SPA** and **Spring Boot** REST APIs secured with **JWT-based authentication**.

## Highlights

- Deployed a Dockerized backend on Render and a Vercel-hosted frontend for a production-ready setup.
- Used **PostgreSQL** for persistence and clean API design for a smooth UX.

## Tech stack

- **Frontend**: React (Vite)
- **Backend**: Java, Spring Boot, JWT
- **Database**: PostgreSQL
- **Deploy**: Docker, Vercel, Render
