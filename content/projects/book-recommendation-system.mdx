## Overview

Designed and built a full-stack recommendation system that generates personalized suggestions using **content-based filtering**, then shipped it via Docker and AWS services.

## Key work

- Implemented the recommendation pipeline and exposed it through web APIs.
- Containerized the app for consistent local/dev/prod behavior.
- Deployed to AWS with storage and compute services for scalability.

## Tech stack

- **Backend**: Python, Django, Flask
- **Data**: MongoDB
- **Infrastructure**: Docker, AWS (S3, ECS, ECR, EC2)
