## Overview

Built backend services to support core e-commerce operations, focusing on **accuracy**, **reliability**, and **performance** for high-traffic workflows.

## What I built

- Real-time inventory management services to keep stock levels in sync.
- Cart pickup flows designed for smooth POS and store-side coordination.
- Observability-friendly APIs (clear contracts, logs/metrics hooks) to reduce production debugging time.

## Tech stack

- **Backend**: Java, Spring Boot
- **Data**: MySQL
- **Search/Monitoring**: Elasticsearch, Dynatrace
